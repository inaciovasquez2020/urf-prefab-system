Terminology

Prefab
A composable structural unit with explicit assumptions, declared invariants, and a bounded scope.

Assumption
An explicitly named dependency or hypothesis required for validity.

Invariant
A property that must hold for a prefab independently of downstream semantics.

Interface
The declared inputs and outputs through which a prefab composes with others.

Scope
The set of claims the prefab explicitly does not make.

Certification hook
A structural attachment point for downstream URF-SG or audit layers.
